# Web-Service
## FrontEnd
### Framework
- React, Vue.js 등
- 비교적 쉽게 개발 가능한 React 활용
- npm; node pakage manager
  - *** dependency issue가 너무 많다 (오래된 소스일 수록)
  - yarn을 통해서 문제해결
  - ```
    $ npm install -g yarn
    $ yarn install
    $ yarn start
    ```
  - PORT 설정
  - ```
    $ PORT=% yarn start
    ```
